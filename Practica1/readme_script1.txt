Las entradas para el script1 son simplemente dos cadenas de caracteres las cuales representan el nombre del jugador uno
y el nombre del jugador dos, el programa las solicita una por una cuando se inicia el script, a partir de aquí el
usuario solo tiene que elegír cuál de los dos jugadores será el vencedor del punto que se está jugando, se necesita dar
como entrada un número entero el cúal deberá ser 1 si se quiere que el punto lo gane el jugador 1 o un 2 si se quiere
que el punto lo gane el jugador 2.
