Para el uso del arbol binario primero tendremos que crear un objeto de tipo ArbolBinario como lo hago en el método
main, posterior se puede insertar elementos con el método insertar(), realizar búsquedas con el método buscar(),
si se quieren realizar recorridos de pre-orden, in-orden y post-orden se pueden llamar con sus respectivos métodos y
cada uno devolverá una lista con la forma del recorrido. Si se quiere imprimir se necesita un ciclo for que itere sobre
la lista imprimiendo cada elemento como en el ejemplo del main.

El método del caminante tal como se pidió solo pide una lista de caracteres U y D, si no, no se rompe, pero no
contabilizará si sube o baja.
